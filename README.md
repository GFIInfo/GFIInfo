# Replication Package

This package contains the following files:

- `coder_1.xlsx`, `coder_2.xlsx`, `coder_3.xlsx`, `coder_4.xlsx`, `final.xlsx` – Coding records from the four coders and the final agreed coding results, respectively.
- `tab_subcat_presence_posthoc.md` and `tab_subcat_frequency_posthoc.md` – Supplementary tables for the paper.
- `analyze_dimensions.py` and `analyze_categories.py` – Scripts to produce the RQ2 results.
- `prediction.py` – Script to produce the RQ3 results.
- `coding_manual.md` – The coding manual.
